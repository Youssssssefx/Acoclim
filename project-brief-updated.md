## Visual Design Guidelines

### Color Palette
- **Primary**: `#002e7f` (Blue)
- **Accent**: `#9d211f` (Red)
- **Background**: `#ffffff` (White)
- **Text**: `#333333` (Dark gray)
- **Shadows**: `rgba(0, 0, 0, 0.15)`

### Typography
- Headings: Bold, modern sans-serif
- Body: Clean, readable sans-serif
- 3D text effects on hero and key headings
- Optimized font sizes for desktop viewing

### 3D Effects
- Drop shadows for depth
- Layered elements
- Smooth transitions
- Subtle animations
- Enhanced effects for desktop experience

### Animations
- Smooth scroll behavior
- Hover effects on cards
- Fade-in on scroll
- Expand/collapse transitions
- Performance-optimized for desktop browsers

## Device Optimization Strategy
- **Primary Target**: Desktop/Laptop (1920×1080, 1366×768, 1440×900)
- **Secondary Target**: Tablets (iPad, Surface)
- **Minimum Support**: Mobile phones (responsive layout adjusts gracefully)
- **Interaction Design**: Hover effects optimized for mouse/trackpad
- **Layout**: Wider layouts, more horizontal space utilization

---

## Notes
- Prioritize desktop/laptop user experience
- Keep design clean despite 3D effects (not overwhelming)
- Ensure accessibility (readable text, sufficient contrast)
- Test primarily on Chrome, Firefox, Safari (desktop versions)
- Mobile version should maintain brand consistency but can simplify interactions
- Ensure easy content management for photo uploads per service category
